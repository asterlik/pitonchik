# ЗАДАНИЕ 1
---------
#### Задание : Замыкание для получения очередной строки из файла.

Создает функцию-читатель файла построчно, используя замыкание для сохранения состояния файлового дескриптора.
Внешняя функция открывает файл и возвращает внутреннюю функцию
Внутренняя функция при каждом вызове:
Читает следующую строку из файла
Убирает символ переноса
При достижении конца файла закрывает его и возвращает None

# ЗАДАНИЕ 2
---------
#### Задание : Декоратор, который будет логировать вызовы функций.

Реализует декоратор для автоматического логирования вызовов функций и их результатов.
Декоратор logger принимает функцию и возвращает обёртку wrapper
Вызывает исходную функцию, сохраняет результат
Выводит информацию: имя функции, аргументы, возвращаемое значение
Возвращает результат исходной функции

# ЗАДАНИЕ 3
---------
#### Задание : Примените декоратор к замыканию.

Комбинация замыкания (чтение файла построчно) и декоратора logger для логирования.
Декоратор логирует только создание читателя, а не чтение строк
В логе видно: имя функции, аргумент 'test.txt', возвращённый объект-функцию
@logger декорирует функцию reader, логируя её вызов и возврат
reader открывает файл и возвращает внутреннюю функцию next_line
next_line при каждом вызове читает и возвращает очередную строку (или None)
