# ЗАДАНИЕ 1
---------
#### Задание : По своему варианту задания создайте пакет, содержащий 3 модуля, и подключите его к основной программе. Основная программа должна предоставлять:
1) Графический пользовательский интерфейс с возможностями ввода требуемых параметров и отображения результатов расчёта,
2) Возможность сохранить результаты в отчёт формата .doc или .xls (например, пакеты python-docx и openpyxl).

**1. Пакет `geometry_pkg` содержит 3 модуля:**
- `parallelepiped.py` – расчёт объёма, площади поверхности и массы прямоугольного параллелепипеда.
- `tetrahedron.py` – расчёт объёма, площади поверхности и массы правильного тетраэдра.
- `sphere.py` – расчёт объёма, площади поверхности и массы шара.

**2. Основная программа `main.py` предоставляет графический интерфейс (tkinter):**
- Выпадающий список для выбора тела (параллелепипед, тетраэдр, шар).
- Динамические поля ввода размеров (см), которые меняются в зависимости от выбранного тела.
- Выпадающий список для выбора материала (алюминий, сталь, пластик) с известной плотностью (г/см³).
- Кнопка «Рассчитать» – вычисляет объём (см³), площадь поверхности (см²) и массу (г).
- Текстовое поле для вывода результатов.
- Кнопка «Сохранить в .docx» – сохраняет результаты расчёта в документ Word.

**3. Дополнительные возможности:**
- Обработка ошибок при вводе нечисловых значений.
- Сохранение последнего расчёта во временную переменную для последующей записи в файл.
- При сохранении открывается диалог выбора имени и расположения файла.
- В сгенерированном отчёте .docx содержатся: название тела, параметры, материал, плотность, объём, площадь поверхности, масса в граммах и килограммах.
