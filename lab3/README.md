# ЗАДАНИЕ 2
---------
#### Задание : Функция для распаковки списка, содержащего другие объекты без рекурсии.

Реализует распаковку вложенных структур без рекурсии, используя стек (LIFO).
Последовательно обрабатывает элементы, раскладывая коллекции обратно в стек
Для словарей добавляет ключи и значения отдельно
В конце переворачивает результат ([::-1]) для сохранения порядка

В отличии от рекурсивной версии:
Не имеет ограничений по глубине вложенности 
Использует явный стек вместо стека вызовов
