# ЗАДАНИЕ 1
---------
#### Задание : Андрей составляет 6-буквенные коды из букв А, Н, Д, Р, Е, Й. Буква Й может использоваться в коде не более одного раза, при этом она не может стоять на первом месте, на последнем месте и рядом с буквой Е. Все остальные буквы могут встречаться произвольное количество раз или не встречаться совсем. Сколько различных кодов может составить Андрей?
1. Импортируется модуль itertools.
2. Задаётся строка lett с буквами А, Н, Д, Р, Е, Й.
3. Счётчик cnt устанавливается в 0.
4. Генерируются все возможные комбинации длиной 6 символов из букв lett.
5. Каждая комбинация превращается в строку s.
6. Проверяется, встречается ли буква 'Й' больше одного раза. Если да — комбинация пропускается.
7. Проверяется, стоит ли 'Й' на первом или последнем месте. Если да — пропуск.
8. Проверяется, есть ли рядом буквы 'Е' и 'Й' (сочетания 'ЕЙ' или 'ЙЕ'). Если да — пропуск.
9. Если все проверки пройдены, счётчик cnt увеличивается на 1.
10. После перебора всех комбинаций выводится итоговое значение счётчика.

# ЗАДАНИЕ 2
---------
#### Задание : Сколько единиц содержится в двоичной записи значения выражения $`8^{2020}+4^{2017}+26−1`$?
1. Вычисляется значение выражения и сохраняется в переменную n.
2. Число n преобразуется в двоичную строку с помощью bin(n), от которой отбрасываются первые два символа (0b), остаётся только двоичная запись.
3. Методом .count('1') подсчитывается количество единиц в двоичной строке.
4. Результат (количество единиц) выводится на экран.

# ЗАДАНИЕ 3
----------
#### Задание : Найдите среди целых чисел, принадлежащих числовому отрезку [245 690; 245 756] [245 690; 245 756] [245 690; 245 756] простые числа. Выведите на экран все найденные простые числа в порядке возрастания, слева от каждого числа выведите его порядковый номер в последовательности. Каждая пара чисел должна быть выведена в отдельной строке. Например, в диапазоне [5; 9] [5; 9] [5; 9] ровно два различных натуральных простых числа  — это числа 5 и 7, поэтому для этого диапазона вывод на экран должен содержать следующие значения: 1 5 3 7
1. Определяется функция is_prime(num), которая проверяет, является ли число простым:
   - Если число меньше 2, возвращается False.
   - В цикле проверяются все делители от 2 до квадратного корня из числа (включительно).
   - Если находится делитель, на который число делится без остатка, возвращается False.
   - Если ни одного делителя не найдено, возвращается True.
2. Счётчик ind устанавливается в 0 для подсчёта порядковых номеров найденных простых чисел.
3. В цикле перебираются все целые числа от 245690 до 245756 включительно.
4. Для каждого числа вызывается функция is_prime().
5. Если число простое:
   - Счётчик ind увеличивается на 1.
   - Выводится порядковый номер и само простое число в одной строке.
