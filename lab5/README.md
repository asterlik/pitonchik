# ЗАДАНИЕ 1
---------
#### Задание : Генератор простых чисел. Просуммируйте возвращаемые числа. К генератору должна быть применена хотя бы одна из функций map, reduce, filter.

1. Функция is_prime(n) определяет, является ли число простым (от 2 и выше).
2. Prime_gen(limit) выдаёт последовательно все простые числа от 2 до заданного предела.
3. Создаётся генератор для предела 160. Первые два числа (2 и 3) выводятся на печать.
4. С помощью reduce вычисляется сумма всех оставшихся простых чисел в генераторе (от 5 до 160 включительно). Результат выводится на экран.
